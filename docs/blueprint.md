# **App Name**: hnosgomeznovopinturas

## Core Features:

- Hero Section: Full-width hero image with a painter at work and a dark gradient overlay. Includes a title and subtitle to capture user attention.
- Service Cards: Display of the services offered (Pintura Profesional, Trabajos Verticales, Reformas Rápidas, Mantenimiento) in a responsive grid layout with hover effects to engage users.
- Image Gallery: A masonry grid layout to showcase 12 images of real painting jobs. Opens images in a lightbox.
- Contact Form: Contact form allowing potential customers to send inquiries. Integrates with Google Sheets/EmailJS.
- Google Maps Integration: Embedding a Google Maps view to show the business location in Coria del Río, Sevilla.
- WhatsApp Chatbot: Implements a chatbot using the WhatsApp API.
- SEO Optimization: The webpage should incorporate features like meta OpenGraph, Schema.org LocalBusiness and a sitemap to be more easily crawled and categorized by search engines.

## Style Guidelines:

- Primary color: #E74C3C (red paint) to convey vibrancy and the brand's core service.
- Background color: #F8F9FA for a clean, light backdrop that emphasizes content.
- Accent color: #2C3E50 (steel blue) as an accent for a modern, industrial feel that complements the primary color.
- Body and headline font: 'Montserrat' (sans-serif) for a clean and modern architectural look, as requested by the user.
- Brutalist-style layout with full-width sections and cards, providing a bold and structured visual experience.
- Subtle fade-in effects on scroll and staggered card animations to add a touch of dynamism and user engagement.
