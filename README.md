# HGNPINTURAS 🎨 & Trabajos Verticales

Sitio web profesional desarrollado para **HGNPINTURAS**, empresa líder en el sector de la pintura profesional y servicios especializados en trabajos verticales en la provincia de Sevilla.

![HGNPINTURAS Banner](https://hgnpinturas.ivangonzalez.cloud/_next/image?url=%2Fimages%2Fassets%2Fhero-vertical.webp&w=1920&q=75)

## 🚀 Sobre el Proyecto

Esta plataforma ha sido diseñada con un enfoque en la conversión y el rendimiento, proporcionando una interfaz moderna y robusta que destaca la experiencia de más de 15 años de la empresa. El objetivo principal es facilitar a los clientes la solicitud de presupuestos y la visualización de proyectos realizados.

### Características Clave:
- **Diseño Premium**: Estética moderna utilizando una paleta de colores corporativa potente (Naranja/Antracita).
- **Optimización SEO**: Estructura de encabezados y metadatos optimizada para posicionamiento local en Sevilla.
- **Rendimiento Máximo**: Puntuaciones de 90+ en Lighthouse para dispositivos móviles.
- **Interacciones Fluidas**: Animaciones suaves con Framer Motion para una experiencia de usuario superior.
- **Responsive Design**: Adaptabilidad total desde smartphones hasta pantallas 4K.

## 🛠️ Stack Tecnológico

- **Frontend**: [Next.js](https://nextjs.org/) (App Router)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Iconografía**: [Lucide React](https://lucide.dev/)
- **Despliegue**: VPS con Coolify / Docker

## 🏗️ Servicios Destacados

1. **Pintura Profesional**: Interiores, exteriores y acabados decorativos de alta calidad.
2. **Trabajos Verticales**: Rehabilitación de fachadas sin necesidad de andamios, optimizando costes y tiempos.
3. **Limpieza en Altura**: Mantenimiento especializado de superficies de difícil acceso.
4. **Tratamientos Especiales**: Impermeabilizaciones y protecciones industriales.

## 📦 Instalación y Desarrollo

Para levantar el proyecto en local:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/jukk4p/hgnpintus.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.
