## 📋 Sobre el Proyecto

**HGN Pinturas (Hermanos Gómez Novo)** es una solución digital diseñada para profesionalizar la presencia web de servicios de pintura técnica y trabajos verticales. El enfoque principal es la **confianza**, la **seguridad** y la **claridad visual**, utilizando una estética *Brutalistas* moderna que destaca la robustez de los servicios ofrecidos.

> [!IMPORTANT]
> El proyecto está optimizado para SEO local en Sevilla y provincia, con un enfoque agresivo en la velocidad de carga y conversión de clientes.

---

## ✨ Características Principales

- 🧗 **Enfoque Vertical**: Secciones dedicadas a trabajos de difícil acceso mediante cuerdas (Rope Access).
- 🛠️ **Gestión de Servicios**: Catálogo interactivo de servicios técnicos (impermeabilización, epoxi, fachadas).
- 📱 **Mobile First**: Experiencia impecable en dispositivos móviles para captación de clientes en ruta.
- ⚡ **Rendimiento Extremo**: Optimización de imágenes y componentes para un Core Web Vitals impecable.
- 📧 **Presupuesto Inteligente**: Formulario de contacto con selector de servicios para pre-cualificación de leads.

---

## 🎨 Diseño Visual

El diseño utiliza una paleta de colores de alto contraste:
- **Primario**: Antracita Profundo (`#111827`)
- **Acento**: Naranja Vibrante (`#D4581A`)
- **Tipografía**: Bebas Neue para títulos (fuerza y visibilidad) e Inter para lectura técnica.

## 🛠️ Stack Tecnológico

| Tecnología | Propósito |
| :--- | :--- |
| **Next.js 15** | Framework de React con App Router para SSR y SEO óptimo. |
| **Tailwind CSS** | Estilizado brutalista y responsive mediante utilidades. |
| **Framer Motion** | Animaciones de revelado y transiciones de página suaves. |
| **Lucide React** | Sistema de iconos vectoriales consistentes. |
| **Shadcn UI** | Componentes de interfaz accesibles y personalizables. |

## 🚀 Instalación y Despliegue

### Entorno de Desarrollo
```bash
# 1. Clonar
git clone https://github.com/jukk4p/HGNPinturas.git

# 2. Instalar
npm install

# 3. Ejecutar
npm run dev
```

### Producción (Docker)
Este proyecto está listo para ser desplegado en contenedores:
```bash
docker build -t hgn-pinturas .
docker run -p 80:80 hgn-pinturas
```
