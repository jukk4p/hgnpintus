# Proyecto: Web de Pintura y Trabajos Verticales 🎨

Repositorio del desarrollo de un sitio web para el sector de la pintura profesional y servicios de trabajos verticales (rehabilitación de fachadas y altura).

![Project Banner](https://hgnpinturas.ivangonzalez.cloud/_next/image?url=%2Fimages%2Fassets%2Fhero-vertical.webp&w=1920&q=75)

## 🚀 Sobre el Proyecto

Este proyecto consiste en el diseño e implementación de una plataforma web orientada a la presentación de servicios técnicos y gestión de presupuestos. El enfoque principal ha sido crear una herramienta digital funcional y con buen rendimiento web.

### Objetivos Técnicos Cumplidos:
- **Diseño Moderno**: Interfaz adaptada a la identidad visual del sector (Naranja/Antracita).
- **Estructura SEO**: Implementación de metatags y jerarquía de contenidos para posicionamiento en buscadores.
- **Rendimiento**: Optimización de carga de recursos para dispositivos móviles.
- **Interactividad**: Uso de librerías de animación para mejorar la navegación del usuario.
- **Adaptabilidad**: Diseño completamente responsive.

## 🛠️ Stack Tecnológico

- **Frontend**: [Next.js](https://nextjs.org/) (App Router)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Iconografía**: [Lucide React](https://lucide.dev/)
- **Infraestructura**: Despliegue mediante contenedores.

## 🏗️ Áreas del Sector Cubiertas en la Web

1. **Pintura**: Visualización de servicios de interior, exterior y acabados.
2. **Trabajos en Altura**: Sección dedicada a la rehabilitación de superficies mediante acceso por cuerdas.
3. **Mantenimiento**: Presentación de servicios de limpieza y protección industrial.

## 📦 Instalación y Desarrollo

Para levantar el proyecto en local:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/jukk4p/hgnpintus.git
   ```
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abrir [http://localhost:3000](http://localhost:3000) en el navegador.
